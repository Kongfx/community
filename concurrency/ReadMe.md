## 选择排序
- 时间复杂度 O n2
- 执行逻辑
~~~
每次从第 i 个数 与 i 后面的每个数比较 如果大于后者则交换 则前 i 个数有序
直到循环结束
~~~

## 冒泡排序
- 时间复杂度 O n2
- 执行逻辑
~~~
从第 i 个元素开始 每次比较 0 到 arr.length - 1 - i 区域中的每两个相邻元素
如果前一个大 则交换
每次外层循环 将产生一个最大值
直到循环结束

~~~
